## 도움말 기준 원칙

- 모든 도움말은 **실제 코드 로직**을 그대로 반영해야 함

## 신뢰도 (confidence)

```
퀀트 점수 기반 기준 신뢰도:
  • 매수(60~100점): 0.50 → 0.95 선형 스케일
  • 매도(0~40점): 0.50 → 0.95 선형 스케일
  • 중립(40~60점): 50점=0.65, 경계=0.50
LLM·퀀트 신호 일치 → ×1.15 (최대 0.95)
한쪽이 중립 → ×0.70 (최소 0.35)
완전 반대(매수↔매도) → 0.35 고정
```

## 목표가

```
AI 리포트에서 추출한 12개월 기준 목표주가입니다.
① 방향 검증 — 매수 신호인데 목표가가 현재가 이하면 폐기
② 범위 검증 — 현재가 대비 −60%~+200% 이탈 시 폐기
③ 컨센서스 대조 — 월가 평균과 60% 이상 괴리 시 두 값의 평균으로 자동 보정
목표가 산출 불가 시 월가 애널리스트 평균 컨센서스로 대체됩니다.
```

## 손절가

```
14일 ATR(평균 실질 변동폭) × 2배를 현재가에서 차감하여 산출합니다.
변동성이 클수록 손절 폭이 넓어지고, 안정적인 종목은 좁게 설정됩니다.
ATR 데이터 없을 경우 현재가 −18%를 기본값으로 사용합니다.
매수(buy) 신호에만 적용됩니다.
```

## 퀀트 세부 — 밸류에이션 (20점)

```
만점 20점 | Greenblatt Magic Formula 기반
EV/EBITDA (10점): 자본구조·세율 왜곡 없는 절대 저평가 지표
  일반: ≤8배→10점 / ≤12배→8점 / ≤16배→6점 / ≤22배→3점
  유틸리티·에너지: ≤12배→10점 / ≤18배→8점 / ≤25배→6점 / ≤35배→3점
  (인프라 투자 특성상 배수가 높은 게 정상 — 기준 완화)
FCF 수익률 (6점): 주주 실질 현금 수익률
  >7%→6점 / >4%→5점 / >2%→3점 / >0%→1점
Forward PER (4점): 컨센서스 이익 기준 상대 저평가
  ≤12배→4점 / ≤17배→3점 / ≤22배→2점 / ≤30배→1점
```

## 퀀트 세부 — 재무건전성 (20점)

```
만점 20점 | Piotroski F-Score + Altman Z-Score 개념
순부채/FCF 상환연수 (8점): 순현금→8점 / ≤2년→6점 / ≤4년→4점 / ≤7년→2점
Debt/Equity (7점): ≤20%→7점 / ≤50%→5점 / ≤100%→3점 / ≤200%→1점
FCF 품질 (5점): FCF>0→+3점, FCF≥순이익×80%→+2점(이익의 질 우수)
※ 금융·부동산 섹터는 D/E·순부채 항목 중립 처리
  (레버리지가 사업 특성상 높은 게 정상이므로 불이익 없음)
```

## Streamlit 모바일 대응 현황

CSS media query로 컬럼을 세로 쌓기 시도했으나 Streamlit 인라인 스타일로 인해 동작 안 함.  
현재 앱 최상단에 모바일 UX 개선용 CSS는 있지만 컬럼 레이아웃 변경은 불가.  
→ **Streamlit 한계로 수용, 더 이상 시도하지 않음.**
